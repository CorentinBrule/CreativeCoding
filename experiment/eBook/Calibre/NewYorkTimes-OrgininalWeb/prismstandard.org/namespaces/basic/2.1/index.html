PRISM Namespace (prism:) Namespace URI = http://prismstandard.org/namespaces/basic/2.1/

This is the PRISM Basic namespace defined in the PRISM SPEC (Version 2.1 errata 2010), and is shared across PRISM implementations.

The PRISM Specifications are developed by the PRISM Working Group as part of the IDEAlliance PRISM Metadata Activity.

For more information about PRISM please visit the PRISM Website.
